# 日本热点排行榜 APP - 产品需求文档

## 这个产品是什么

一个 PWA 应用（可以添加到手机主屏当 APP 用，也可以在电脑浏览器里打开），自动从 Instagram、X(Twitter)、Yahoo! Japan、TikTok 四个平台收集日本区的热门内容，生成中文排行榜。参考微博热搜的形式，总榜 100 名，分类榜各 50 名。每条热点贴出 5 条原帖链接作为依据。

功能就这些，不做用户画像、不做营销建议、不做趋势分析。

---

## 页面和功能

整个 APP 就 2 个页面：

### 页面 1：排行榜（首页）

**顶部**：APP 名称 + 最近更新时间

**分类标签栏**（可横向滑动）：
`总榜 | 娱乐 | 动漫 | 时尚 | 社会 | 科技 | 生活 | 萌宠`

**排行榜列表**：
- 总榜显示 100 条，分类榜各显示 50 条
- 每条热点显示：
  - 排名序号（1、2、3...）
  - 热点标题（中文翻译），标题下一行小字显示日文原文
  - 热度标签：`爆`（红色，短时间飙升）/ `热`（橙色，持续高热度）/ `新`（蓝色，刚上榜）
  - 来源平台图标（X / Yahoo / TikTok / Instagram，一个热点可能有多个平台图标）

**点击某条热点**：展开显示该热点的 5 条原帖（不跳转新页面，直接在当前位置展开/收起），每条原帖显示：
  - 来源平台图标 + 发帖人昵称
  - 帖子内容摘要（中文翻译，最多 2 行）
  - 互动数据：点赞数 / 转发数 / 评论数
  - 可点击的"查看原帖"按钮，跳转到对应平台的原始帖子

**前 3 名特殊样式**：排名 1-3 的热点用更大字体或高亮背景突出显示。

### 页面 2：热点详情页

从排行榜点击热点标题（不是展开原帖，而是点标题本身）进入详情页：

- 热点标题（中文 + 日文）
- 中文摘要（3-5 句话说清楚这个热点是怎么回事）
- 来源平台 + 热度标签
- 5 条原帖完整展示（和排行榜展开看到的一样，但这里显示完整内容不截断）

## 数据来源

| 平台 | 抓什么 |
|------|--------|
| X (Twitter) | 日本区 Trending 话题 + 话题下热门推文 |
| Yahoo! Japan | 实时搜索热词排行 + 相关新闻 |
| TikTok | 日本区热门标签 + 高互动视频 |
| Instagram | 日本区热门标签 + 高互动帖子 |

不再使用 5ch。

## 数据处理流程

```
用户手动点击"刷新"按钮触发：
抓取 4 个平台热点 → AI 翻译成中文 → 相同话题合并 → 算热度排名 → AI 自动分类 → 每个热点取 5 条最热原帖 → 存数据库 → APP 展示
```

热度排名算法：
- 互动总量（点赞+转发+评论）占 40%
- 增长速度（单位时间内互动增量）占 30%
- 跨平台程度（多个平台都在讨论则加分）占 20%
- 内容质量（有权威媒体或大V参与）占 10%

## 数据库表设计

只需要 2 张表：

### trending_topics（热点主表）

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 唯一编号 | 1 |
| title_ja | 日文标题 | 大谷翔平がホームラン |
| title_zh | 中文标题 | 大谷翔平打出全垒打 |
| category | 分类 | 体育 |
| heat_score | 热度分数 | 95 |
| heat_tag | 热度标签：爆/热/新 | 爆 |
| sources | 来源平台 | ["x", "yahoo", "tiktok", "instagram"] |
| summary_zh | 中文摘要 | 大谷翔平今日比赛中打出赛季第30号全垒打... |
| rank_overall | 总榜排名（1-100，null 表示未上总榜） | 1 |
| rank_category | 分类榜排名（1-50） | 1 |
| created_at | 首次上榜时间 | 2026-04-01 10:00 |
| updated_at | 最近更新时间 | 2026-04-01 10:30 |

### topic_posts（原帖，每个热点 5 条）

| 字段 | 说明 | 示例 |
|------|------|------|
| id | 唯一编号 | 1 |
| topic_id | 关联热点 | 1 |
| platform | 来源平台 | x |
| author_name | 发帖人昵称 | @shohei_fan |
| content_ja | 原文摘要 | 大谷翔平すごい！ |
| content_zh | 中文翻译摘要 | 大谷翔平太厉害了！ |
| likes | 点赞数 | 12000 |
| reposts | 转发数 | 3500 |
| comments | 评论数 | 800 |
| post_url | 原帖链接 | https://x.com/xxx/status/123 |
| posted_at | 发帖时间 | 2026-04-01 09:30 |

## 开发顺序

1. **搭界面**：用假数据做出排行榜页面 + 详情页 + 展开原帖交互，配置 PWA
2. **接数据库**：连 Supabase，建表，手动加数据验证页面能正常显示
3. **自动抓数据**：写定时任务从 4 个平台抓热点和原帖，AI 翻译+分类+摘要，存入数据库
4. **部署上线**：部署到 Vercel，手机添加到主屏测试 PWA 效果
